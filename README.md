# read4me-api

### Read4me PDF to voice converter API

##### 1. Requirements:

* [espeak](https://github.com/rhdunn/espeak)
* [PoDoFo](https://github.com/mekentosj/podofo)
    
##### 2. Using:

##### 2.1. espeak:

* Compile espeak:
    
    `cd app && make`
        
* Start espeak:
        
    `espeak -v pt -s 60 "Example of speech using Brazilian Portuguese language"`
